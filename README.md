# API Key Tester

A simple web app to test your API keys for different AI providers. Perfect for beginners who want to check if their keys work before using them in projects.

## What This App Does

- **Test API Keys**: Paste your API key and see if it's valid
- **Multiple Providers**: Supports OpenRouter, Google Gemini, and more
- **Easy to Use**: Just enter your key and click "Test"
- **Free to Use**: No cost to test your keys

## Quick Start

### 1. Install Dependencies
```bash
bun install
```

### 2. Run the App
```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### 3. Test Your Keys
- Go to the homepage
- Select your provider (OpenRouter, Google Gemini, etc.)
- Paste your API key
- Click "Test Key"
- See the results!

## Getting API Keys

### Google Gemini (Free!)
1. Go to [makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)
2. Sign in with Google
3. Create a new API key
4. Copy and paste it into the app

### OpenRouter
1. Go to [openrouter.ai](https://openrouter.ai)
2. Sign up for an account
3. Get your API key from the dashboard
4. Note: Free tier has limits

### Other Providers
- **Mistral**: [mistral.ai](https://mistral.ai) - Free API
- **Together AI**: [together.ai](https://together.ai) - Free credits for new users
